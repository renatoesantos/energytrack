# EnergyTrack ⚡

Sistema IoT para monitoramento de consumo de energia.

## Tecnologias

Frontend

- React
- React Router DOM
- JavaScript (ES6+)
- CSS

Backend

- Node.js
- Express
- Socket.io

Banco de Dados

- PostgreSQL

## Como rodar

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
